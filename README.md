## 📊 Sobre o Projeto

O **K'PY AUTOMATE** é um sistema web profissional para análise de dados comerciais de supermercados/redes de varejo, focado em processar relatórios do sistema SGE (Sistema de Gestão Empresarial).

### ✨ Funcionalidades

| Módulo | Descrição |
|--------|-----------|
| 📈 **Curva ABC** | Análise de vendas por loja com filtros e exportação |
| 📦 **Entradas** | Análise de entradas por grupo com mapeamento de compradores |
| 📉 **Produtos sem Venda** | Processamento de relatórios Sai23 (múltiplos arquivos) |
| 📋 **Relatório de Ruptura** | Combinação de Estoque + Curva ABC + Média de Vendas |

### 🎯 Diferenciais

- ✅ Upload de múltiplos arquivos Excel (.xlsx, .xls)
- ✅ Filtros dinâmicos por loja, grupo, comprador
- ✅ Exportação para Excel com formatação brasileira
- ✅ Barra de progresso e loading skeletons
- ✅ Animações suaves e interface moderna
- ✅ 100% client-side (funciona sem servidor)

### Como usar cada funcionalidade

#### 📊 Curva ABC
1. Clique em "Curva ABC" no menu
2. Arraste ou selecione o arquivo Excel da Curva ABC
3. Clique em "Processar"
4. Filtre por lojas usando os botões de macro (EMPÓRIO, MERCEARIA, OUTROS)
5. Exporte o resultado para Excel

#### 📦 Entradas
1. Clique em "Entradas" no menu
2. Selecione o arquivo "Entradas Produtos por Grupo"
3. Processe e filtre por grupo ou comprador
4. Exporte os dados

#### 📉 Produtos sem Venda
1. Clique em "Produtos s/ Venda" no menu
2. Selecione um ou mais arquivos Sai23
3. Processe, filtre por loja e exporte

#### 📋 Relatório de Ruptura
1. Clique em "Criar Relatório" → selecione "RUPTURA"
2. Carregue os arquivos de Estoque e Curva ABC
3. Gere o relatório combinado
4. Use "Varrer" para limpar dados (matriz, NC, zerados)
5. Filtre por lojas e exporte
